## 実行方法
###  VSCode の "Code Runner" を使う方法
コマンドライン引数が無い場合はこちらが便利
1. extentions(4つのブロックがあり、右上だけ斜めになっているもの)から "Code Runner" をインストール。
2. 右上あたりに横向きの正三角形が現れるので、作成したプログラムを表示した状態でクリック

### コマンドでコンパイル・実行する方法
1. ターミナル（コマンドプロンプト）を起動し、作成したプログラムのディレクトリまで cd コマンドで移動
2. 以下のコマンドでコンパイル
```
gcc hoge.c -o hoge
```
- hoge.c: コンパイルしたいソースコード
- hoge: 実行ファイルの名前

3. プログラムの実行
コンパイルが正常にできたら、以下のコマンドで実行
```
./hoge
```
