# アルゴリズムとデータ構造2（ALgorithm and DataStructure 2）

## ディレクトリ構成
- merge-sort: マージソート関連
    - merge.c: マージ処理自体のプログラム
    - merge-sentinel.c: 番兵を利用したマージ処理のプログラム
    - merge-sort.c: マージソートのプログラム
    - merge-sort-simulation.c: ランダムに生成した数がマージソートでソートされる過程を可視化するプログラム
- quick-sort: クイックソート関連
    - quick-sort.c: クイックソートのプログラム
- list: リスト構造
    - list_sample.c: リスト構造の動作を検証するプログラム
    - list.c: リスト構造とその処理を定義するプログラム
    - list.h: リスト構造とその処理の宣言をするヘッダファイル
